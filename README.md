# 💎 Diamond Price Prediction

A Machine Learning project that predicts the **price of a diamond based on its physical and quality attributes** using regression techniques.  
The model is trained on the **Kaggle Diamonds Dataset** and aims to demonstrate the complete machine learning workflow including **data exploration, feature engineering, model training, and evaluation**.

# 📖 Project Motivation

Diamonds are priced based on several attributes such as **carat, cut, color, clarity, depth, and table**.  
However, determining the correct price manually can be difficult.

This project demonstrates how **Machine Learning regression models** can estimate diamond prices based on these features.

The goal of the project is to:

- Apply **data analysis and machine learning techniques**
- Understand how different diamond attributes influence price
- Build a **predictive model**
- Demonstrate a **complete ML workflow suitable for portfolio projects**

---

# 📊 Dataset

This project uses the **Diamonds dataset from Kaggle**.

📎 Dataset Link:  
https://www.kaggle.com/datasets/shivam2503/diamonds

### Dataset Features

| Feature | Description |
|------|-------------|
| carat | Weight of the diamond |
| cut | Quality of the cut |
| color | Diamond color grading |
| clarity | Diamond clarity grading |
| depth | Total depth percentage |
| table | Width of top of diamond |
| price | Price in USD |

The dataset contains **over 50,000 diamond records**.
